For initial configuration:
	- Please, rename .env.example to .env and change the variable values with your own data.
	- You need MySQL running in your PC. If not installed, please install it first.
	- If installed (MySQL) but still don't running properly, please check in Task Manager if mysql is actually running.


For generating the .exe file:
	- Please, run in your terminal: pyinstaller main.spec to create an .exe file
